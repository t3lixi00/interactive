1.Definitions

HTML: Hyper Text Markup Language, is the container for Web pages. whereas hyper text means that non-text element like pictures, links, music, or programme and so on.

CSS:Cascading Style Sheets, is a style sheet language used for decorating web pages and animating them.

JAVASCRIPT: is an object-oriented computer programming language commonly used to create interactive effects within web browsers.

HTTP: is Hypertext Transfer Protocol, the set of rules for transferring files on the world wide web.

HTML5: the fifth version of HTML standard. The main difference is adding new elements, new types of form control and so on.

DHTML: Dynamic HyerText Markup Language, is a combination of Web development technologies used to create dynamically changing websites. Web pages may include animation, dynamic menus and text effects
