1. What you can do with the jQuery? How it can help you as web developer?

It helps developers to to create animations, to change element CSS, select DOM elements, develop Ajax applications. Actually, It simplifies the javascript, which is faster and light weight than javascript . And it offers a wide range of plugins for meet various specific needs

3. What are CDNs? What kind of benefits you get if you use them?

It is a content delivery network, which means that, a servers’ system provides webpages or javascript library, bootstrap library and so on. This way is faster and safer for website. Because servers will choose closer server to send response to customers, and if one server is not working, others server will still offer the library.
